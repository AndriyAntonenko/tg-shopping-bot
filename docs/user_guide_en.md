# User Guide

Welcome to the **TG Shopping Bot**! This bot allows you to browse products, place orders, and manage your shopping experience directly within Telegram.

## Getting Started

To start using the bot, simply send the `/start` command. You will receive a welcome message and a menu with available options.

## Commands

Here is a list of available commands:

- `/start` - Start the bot and see the main menu.
- `/help` - Show the help message with a list of commands.
- `/catalog` - Browse the product catalog.
- `/orders` - View your pending orders (Admin only: View all pending orders).
- `/feedback` - Send feedback to the administrators.
- `/language` - Change the bot's interface language.
- `/admin` - Access the admin panel (Admin only).
- `/add_item` - Add a new product to the catalog (Admin only).
- `/remove_item` - Remove a product from the catalog (Admin only).

## For Administrators

### Managing Orders
- Use `/orders` to view a list of all pending orders.
- You can **Approve** or **Cancel** orders directly from the order details view.

### Managing Products
- **Add Product**: Use `/add_item` to add a new product. You will be prompted to enter the name, description, price, and upload an image.
- **Remove Product**: Use `/remove_item` to delete a product from the catalog.

### Viewing Feedback
- Use the **Feedbacks** button in the admin panel to view user feedback.
